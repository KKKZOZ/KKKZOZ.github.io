一般是想到什么写什么，逻辑会比较混乱。

总的来说，整个 11 月份还是过的比较自律和高效的，看了许多分布式领域的经典论文，大部分也做了笔记。

最满意的是 11 月份的运动量提上来了，在 11 月份的最后几天成功实现了良性循环：吃完晚饭，六点半去图书馆学习，学到九点半，直接去操场跑步，跑个五六公里的样子，然后回宿舍，先洗澡，再煮两个蛋，看会儿 B 站，就差不多 11 点半了，洗漱一下，就打算上床了。11 月末我还专门买了一个充电小台灯，12 点到 12 点 40 都是我的看书时间，最近把余华的《兄弟》又翻出来看了看。

这样挺好的，把晚上那段经常打游戏的时间“水”过去了，最近跑步也越跑越有意思，应该是没以前这么累了吧，能感觉到一定的正反馈了。

11 月份还看了看其他类型的课外书，比如 Haskell 的入门书籍，简单地领略了下函数式编程的魅力，还有就是《SICP》，希望自己空暇的时候能一直看下去。

我还注意到了一门新兴语言，也不知道以后能不能发展起来，moonbit，语法现阶段和 rust 非常相似，我还提交了两次 pull request 去改善它的中文文档质量来着哈哈。

11 月份我还看了 Epoxy 的源码，说起来这应该是我第一次系统性地去看一个应用的源码，收获还是挺大的，学会了怎么通过 debug 的方式来观察调用路径，以及如何理解应用的整体架构等，特别是在 Java 的并发编程这一方面，为了看懂源码，恶补了一大块这一方面的知识。

看这一方面的知识时，我才意识到 Java 其实也能像 Golang 那样写并发编程的代码，我之前一直不知道在 Java 中怎么实现类似于 channel 的功能，后来才发现，我只是需要一个 BlockingQueue 哈哈，我感觉 community 中应该有包装好的 Channel 类，下次要用时再去找找。

哦对了，还有一件事，恭喜 T1 !!!

希望李哥明年直接成为五冠王!
