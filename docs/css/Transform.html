Translate #

This transform function moves an element sideways, or up and down.

小范围内改动

  • transform: translate(20px, 10px)
  • translateX(value)
  • translateY(value)

注意方向：

  • X轴：左正右负
  • Y轴：下正上负（和数学中的不一样）

It’s important to note that an element using transform will not cause other elements to flow around it.

其他元素不会动，就是会发生重叠，可以考虑用z-index来达到期望中的效果

It’s also worth noting that translate will be hardware accelerated if you want to animate that property, unlike position: absolute.
